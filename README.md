## Getting Your OpenAI API Key

Step 1 — Go to platform.openai.com
Step 2 — Click "Sign Up" and create a free account
Step 3 — Go to "API Keys" section
Step 4 — Click "Create new secret key"
Step 5 — Copy the key (starts with sk-)
Step 6 — When deploying on Streamlit add it to Secrets:
         OPENAI_API_KEY = "paste-your-key-here"

Note: OpenAI charges a small usage fee (~$5-20/month)
paid directly to OpenAI — not to us.
This is standard for all AI chatbot services.

## How To Get It Live (Free)

Step 1 — Create free GitHub account at github.com
